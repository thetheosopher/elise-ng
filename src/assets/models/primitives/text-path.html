The Text Path element renders text positioned and rotated along a guide path, following SVG path commands such as lines, cubic Bézier curves, and arcs.

Create the element with elise.textPath(text, pathData), supplying the text string and an SVG path data string that defines the guide curve.

Notable properties:

  • startOffset – distance along the path where text begins.
  • startOffsetPercent – treat startOffset as a percentage of path length.
  • showPath – render the guide path itself.
  • side – left or right side of the path to lay out text.
  • alignment – left, center, or right along the path.

Typography controls (typeface, typesize, typestyle, letterSpacing, textDecoration) and richText runs work the same as on a standard Text element.

The Text Path element is both strokeable and fillable.
