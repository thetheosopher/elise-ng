The Text element is used to display either embedded text content or text content from an external resource. Its layout rectangle is defined by its location and size properties.

The content property may be used to specify text content directly or the source property may be used to reference a text resource, which may in turn either contain the text content directly or reference an external URI from which the text content should be retrieved.

The typeface property may be used to specify a font name. The typesize property may be used to specify a font size. The typestyle property may be used to control the style of the rendered text. The alignment property may be used to determine text alignment with the layout box from the following options:

  • top
  • bottom
  • left
  • right
  • center
  • middle
  • end
  • start

The Text element is both strokeable and fillable and must either be stroked or filled to be visible.
