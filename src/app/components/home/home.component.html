Design Surface

Author graphics in the browser

DesignController transforms any model into a full editing surface. Selection, rubber-band, drag/resize/rotate, grid snapping, smart alignment guides, clipboard, undo/redo, inline rich-text editing, and 14 creation tools — all built in.

  • Rectangle, ellipse, line, polygon, polyline, pen, arc, arrow, ring, wedge, text, image, model, and regular polygon tools
  • Point editing for path, polyline, polygon, and arc elements
  • Undo/redo, z-order, copy/cut/paste, and aspect-locked sizing
  • Inline caret-based rich-text editing with full keyboard navigation

Surface Framework

Build full applications

Surface provides a layered application container for kiosks, signage, marketing displays, and interactive presentations — combining canvas-rendered content with HTML, image, video, and animated layers.

  • Multi-pane navigation with 20+ built-in transitions
  • Image, HTML, video, and animation layers
  • Buttons, radio strips, text layers, and hidden hit zones
  • Ideal for marketing kiosks with simple declarative navigation
